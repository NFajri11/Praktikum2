## Diketahui rumus luas dan keliling lingkaran
 Luas = Phi x r2(kuadrat) <br />
 Keliling = 2 x Phi x r  <br />
 Nilai Phi yang digunakan adalah 3.14 <br />  
 r merupakan jari jari lingkaran <br />
